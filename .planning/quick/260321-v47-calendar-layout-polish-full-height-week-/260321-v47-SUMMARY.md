## What Was Built

Five targeted changes to `src/routes/calendar.js`:

**1. Two-row control bar:**
- Row 1: page title/subtitle on left; view toggle (Month/Week/Agenda) + gear icon + New Post button grouped on right
- Row 2: funnel filter icon on left; nav arrows (prev/label/next) right-aligned via `ml-auto`
- `id="nav-arrows"` preserved on row 2 so existing JS that hides it in agenda view still works

**2. View toggle spacing:**
- Buttons changed from `px-3 py-1.5 text-xs` to `px-4 py-2 text-sm` for more breathing room
- New Post button updated to match at `px-4 py-2 text-sm`

**3. Collapsible filter bar:**
- Filter chips container (`id="filter-bar"`) now starts with `hidden` class
- Funnel icon button (`id="filter-funnel-btn"`) in row 2 toggles visibility via `filterBar.classList.toggle('hidden')`
- `applyFilters()` still called on page load so localStorage filter state is applied regardless of bar visibility

**4. Full-height week columns:**
- Week view day cells changed from `min-h-[200px]` to `min-h-[calc(100vh-220px)]`
- Columns now fill near full viewport height below the control rows and day headers

**5. All existing functionality preserved:**
- SortableJS drag-and-drop, day panel click, view switching, gear dropdown, filter persistence all unchanged
